# 数据驱动

只分析数据是怎么映射到 dom 的 ？在模版中插入一个变量最终怎么生成 dom 的 数据的变化来驱动视图的变化（响应式原理中分析）

初始化工作，做 uid 传入的参数都 merge 到 $options 上

import Vue from 'vue'
const options = {
	el: '#app',
	data: {
		message: 'hello',
	},
}
// init的时候：传入的参数都 merge 到 $options 上，可以通过vm.$options.el获取传入的el，vm.$options.data获取传入的data
const app = new Vue(options)
